# YouTube Cookies

Поместите сюда файлы cookies в формате .txt для обхода ограничений YouTube.

Как получить cookies:
1. Установите расширение "Get cookies.txt LOCALLY" в браузере
2. Зайдите на youtube.com
3. Экспортируйте cookies в файл .txt
4. Поместите файл в эту папку

Пример: `youtube_cookies.txt`
