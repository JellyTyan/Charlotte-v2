# Data We Collect

### The bot may collect and process the following data:
- Telegram user ID
- Language and basic Telegram metadata
- Messages and links sent to the bot
- Media files required to fulfill user requests (processed temporarily)

### Purpose of Data Collection
Data is used solely to:
- Provide and maintain bot functionality
- Process user requests
- Improve service quality and performance
- Enable optional premium features

### Data Storage
- Media files are stored temporarily and deleted automatically after processing
- Logs and technical data are kept only as long as necessary for maintenance and debugging

### Data Sharing

**Charlotte does not sell or share personal data with third parties.**

### User Rights

Users may request deletion of their data by contacting the developer.
